# 🥁 Interactive Drum Machine

A clean and responsive Drum Machine built using HTML, CSS, and JavaScript.  
Users can play drum sounds using both keyboard keys and mouse clicks with smooth interactive feedback.

---

## 🚀 Features

- Keyboard Support (Q, W, E, A, S, D, Z, X, C)
- Clickable Drum Pads
- Real-time Audio Playback
- Interactive Sound Display
- Responsive Layout
- Minimal and Modern UI

---

## 🛠️ Built With

- HTML5
- CSS3
- Vanilla JavaScript

---
